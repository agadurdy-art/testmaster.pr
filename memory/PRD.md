# IELTS Ace - Product Requirements Document

## Original Problem Statement
IELTS exam preparation platform being transformed into a "Hybrid Learning Platform" controlled by teachers. Phase 1 focus: Vocabulary Engine for Advanced Course.

## Core Architecture
- **Frontend**: React + Tailwind CSS + Shadcn UI
- **Backend**: FastAPI on port 8001
- **Database**: MongoDB
- **LLM**: OpenAI GPT-4o via Emergent LLM Key
- **TTS/STT**: OpenAI TTS (nova) + Whisper via Emergent Integrations

## What's Been Implemented

### Vocabulary Engine - Phase 1 COMPLETE (Feb 12, 2026)
All 5 modules built with iOS-inspired design (`#F5F5F7` system gray, white cards, `rounded-[20px]`, subtle shadows, `backdrop-blur-xl` headers):

1. **Learn Mode** (`/vocabulary/learn/:moduleId`): Full-screen flip-card slides (28/module), IPA, TTS, word formation table, keyboard nav
2. **Controlled Practice** (`/vocabulary/practice/:moduleId`): Fill-in-blank + matching exercises (25/module), hints, score tracking
3. **Mastery Quiz** (`/vocabulary/quiz/:moduleId`): 10 MCQ, 80% pass, question review. Auto-adds wrong answers to Review Bank
4. **Production Mode AI** (`/vocabulary/production/:moduleId`): Write sentences with target words, GPT-4o evaluates grammar & usage (1-5 stars)
5. **Review Bank** (`/review-bank`): Spaced repetition flashcards (1d->3d->7d->14d->mastered). "Got It!" / "Still Learning" buttons

**Backend APIs:**
- `/api/vocabulary-engine/{moduleId}/slides`, `/practice`, `/quiz`
- `/api/vocabulary-engine/quiz/submit` (auto-adds wrong answers to review bank)
- `/api/vocabulary-engine/evaluate-sentence`
- `/api/vocabulary-engine/review-bank/{userId}`, `/add`, `/review`
- `/api/vocabulary-engine/progress`, `/progress/{userId}`
