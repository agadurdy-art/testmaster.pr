# IELTS Ace - Product Requirements Document

## Original Problem Statement
An IELTS exam preparation platform with AI-powered evaluation, Cambridge-aligned test content, and comprehensive practice modes for Reading, Listening, Writing, and Speaking skills.

## Core Architecture
- **Frontend**: React with Tailwind CSS + Shadcn UI
- **Backend**: FastAPI (Python) on port 8001
- **Database**: MongoDB
- **TTS**: OpenAI TTS (nova voice) via Emergent Integrations
- **STT**: OpenAI Whisper via Emergent Integrations
- **LLM**: OpenAI GPT-4o via Emergent LLM Key

## What's Been Implemented

### Liz AI Teacher - Real Teacher Experience (Feb 12, 2026)
**NOT a chatbot - a real IELTS teacher who speaks, guides, and leads lessons.**

#### Teacher Presence UI
- Large centered avatar with status-based animations (idle/thinking/speaking)
- Glowing border + audio visualization bars when speaking
- Auto-greeting with personalized message on page load

#### Voice-First Interaction
- **TTS (auto)**: Every Liz response spoken automatically (OpenAI nova voice)
- **STT**: Student speaks via microphone (OpenAI Whisper)
- Auto-voice toggle in header
- Microphone = primary input, text = secondary

#### Structured Lesson Modes
- Speaking Practice, Vocabulary Builder, Grammar Lesson, Study Plan

#### Progress Awareness (Memory Injection)
Liz knows: name, level, target band, weak/strong skills, last 5 scores, study streak, inactivity, completed lessons, grammar quiz progress

#### Homework System (Feb 12, 2026) - NEW
- **Auto-Assignment**: Liz assigns homework during lessons using [HOMEWORK] markers, auto-parsed by backend
- **Manual Assignment**: POST /api/liz/homework/assign endpoint
- **Submission**: Students submit answers via expandable homework cards
- **Auto-Review**: GPT-4o evaluates submissions with IELTS criteria, score (X/10), specific feedback
- **UI**: Homework icon with pending badge in header, expandable cards with task/feedback/score
- **Greeting Integration**: Liz reminds students about pending/overdue homework
- **Types**: vocabulary, writing, grammar, speaking
- **MongoDB**: `liz_homework` collection

#### Key Endpoints
- `POST /api/liz/greet` - Personalized greeting + TTS + homework reminders
- `POST /api/liz/chat` - Chat with auto-homework detection
- `POST /api/liz/tts` / `POST /api/liz/stt` - Voice
- `GET /api/liz/homework/{user_id}` - List homework
- `POST /api/liz/homework/assign` - Create homework
- `POST /api/liz/homework/{homework_id}/submit` - Submit + auto-review
- `DELETE /api/liz/homework/{homework_id}` - Remove homework
- Session management: new-session, sessions, history
