## Sample Lesson Flow (Unit 3, Lesson 1: Primary Colors)

### 10-Step Flow

**1. Retrieval Warm-up (3 min)**
- Show 3 images from previous unit (numbers)
- "How many apples?" (answer: three)

**2. Vocabulary Input (6 min)**
- Present: red, blue, yellow, color, see, look
- Each word: image + pronunciation + example sentence
- "Red. /red/. This is red."

**3. Micro Game - Vocab (4 min)**
- Matching game: color word ↔ color swatch
- Recognition speed drill

**4. Micro Reading (4 min)**
- Short text (30 words):
  > "Look! I see colors. The apple is red. The sky is blue. The sun is yellow. Colors are beautiful!"
- Highlight vocabulary words

**5. Grammar Focus (3 min)** [Light for Stage 1]
- Pattern: "What color is it?" → "It is [color]."
- Visual examples

**6. Micro Game - Grammar (3 min)**
- Click the correct answer
- "What color is the banana?" → "It is yellow"

**7. Listening Task (5 min)**
- Audio: "Point to red. Point to blue. Point to yellow."
- Student follows instructions

**8. Production - Speaking (5 min)**
- Prompt: Show red apple image
- Student says: "It is red" or "The apple is red"
- Record & playback

**9. Exit Ticket (3 min)**
- 4 quick questions:
  1. What color is this? (show blue)
  2. Match word to color
  3. Listen and choose
  4. Say the color (speaking)
- Pass threshold: 3/4 correct

**10. Auto-Review Scheduled**
- Words added to spaced repetition queue
- Next review: 24 hours

## Visual Asset Strategy Options

### Option A: AI Generation (Fast but Variable)
- Use DALL-E / Midjourney / Nano Banana for all images
- Pro: Fast, scalable, consistent style possible
- Con: May need prompt engineering, occasional odd results

### Option B: Stock Images (Slower but Professional)
- Source from Unsplash, Pexels, or paid stock
- Pro: Professional quality, consistent
- Con: Finding matching style across 300+ images is hard

### Option C: Hybrid Approach (Recommended)
- **Core objects** (animals, food, colors): Stock images (consistency matters)
- **Abstract concepts** (actions, emotions): AI generated
- **Diagrams** (body parts, rooms): Custom illustrations

### Option D: Placeholder First
- Use colored placeholder boxes with word text
- Replace with real images over time
- Pro: Can start building immediately
- Con: Poor initial user experience
